# Booking Widget

An adaptive React-based booking widget for selecting dates and time slots with smooth scrolling functionality.
[Show here](https://amahalias.github.io/6037_book_session/).

## Features

- Selectable date slots with dynamic month display.
- Selectable time slots with disabled past times.
- Smooth scrolling for both date and time slots.
- Customizable styles and reusable components.

## Getting Started

### Prerequisites

- Node.js and npm installed.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/AMahalias/6037_book_session.git
   ```
2. Navigate to the project directory:
   ```bash
   cd booking-widget
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Running the Project

Start the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to view the app.

## Project Structure

- `src/components`: Contains React components.
- `src/helpers`: Reusable utils for managing scroll behavior.
- `src/hooks`: Custom hooks for managing scroll behavior.
- `src/styles/globals.css`: Global styles for the project.

## Customization

- Update colors in `src/styles/globals.css` to match your design.
- Modify components in `src/components` for additional functionality.
